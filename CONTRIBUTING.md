# Contributing

Thank you for your interest in contributing to this project!

## Getting Started

1. Fork the repository
2. Create a feature branch from `main`
3. Make your changes
4. Run `make reviewable test` to verify
5. Submit a pull request

## Development

This provider is built with [Upjet](https://github.com/crossplane/upjet). To generate resources:

```bash
make generate
```

To build:

```bash
make build
```
